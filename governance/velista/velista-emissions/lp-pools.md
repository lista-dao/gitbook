# LP 池

## 合格的 LP 池

以下是用户可以提供流动性以赚取 LISTA 奖励的所有合格池的表格。

| **协议**    | **池名称**      |
| --------- | ------------ |
| Lista DAO | lisUSD 单独质押池 |
| Lista DAO | 借用 lisUSD    |

## LP 池奖励条件

为了有资格获得 LP 池类别下的 LISTA 奖励，用户必须在 Lista DAO 的[奖励](https://lista.org/rewards?utm_source=gitbook\&utm_medium=article\&utm_campaign=lp-pools)部分质押他们的 LP 代币。

此外，不同类型的 LP 必须满足某些条件。

## 内部 LP

参与 Lista DAO 的 DeFi 生态系统的用户将有机会赚取 LISTA 奖励。这些奖励作为社区激励计划中的 LP 池奖励分配。

### 1. lisUSD 借款人

lisUSD 借款人也有资格获得 LISTA 奖励。分配给每个用户的奖励与借出的 lisUSD 金额相比于所有借出的 lisUSD 的总价值锁定（TVL）成正比。

**规则和条件：**

1. 用户必须在 Lista DAO 的[平台](https://lista.org/cdp/loans?utm_source=gitbook\&utm_medium=article\&utm_campaign=lp-pools)上借用 lisUSD。
2. 奖励每秒发放一次，即 lisUSD 代币在 Lista DAO 平台上借出的那一刻开始。

### 2. lisUSD 单独质押池

在 Lista DAO 的 lisUSD 单独质押池中质押其 lisUSD 的用户也有资格获得 LISTA 奖励。分配给每个用户的奖励与质押的 lisUSD 金额相比于 lisUSD 单独质押池的总价值锁定（TVL）成正比。

**规则和条件：**

1. 用户必须在 Lista DAO 的[平台](https://lista.org/cdp/earn?utm_source=gitbook\&utm_medium=article\&utm_campaign=lp-pools)上单独质押 lisUSD。
2. 奖励每秒发放一次，即 lisUSD 代币在 Lista DAO 的 lisUSD 单独质押池中质押的那一刻开始。

## 外部 LP

当用户在 Lista DAO 上质押他们的 LP 代币时，我们将这些代币重新质押到外部 DeFi 平台，如 PancakeSwap 和 Thena Finance。通过这一过程，用户获得双重奖励——他们从 Lista DAO 获得 LISTA 发放以及从 PancakeSwap（以 CAKE 奖励）和 Thena（以 THE 奖励）获得的农场/LP 奖励。

Lista DAO 支持来自流行 DeFi 协议的多种 LP 代币，使用户能够高效地质押并赚取奖励。支持的 LP 代币包括以下内容：

| **协议**        | **池名称**              |
| ------------- | -------------------- |
| Pancake Swap  | lisUSD/USDT 稳定池 LP   |
| Thena Finance | LISTA/USDT 窄 LP      |
|               | slisBNB/BNB 相关 LP    |
|               | lisUSD/FRAX（稳定）LP    |
|               | lisUSD/USDT（cl 稳定）LP |
|               | lisUSD/BNB ICHI LP   |

**规则和条件：**

1. **在 Lista DAO 上质押**：用户可以通过[此链接](https://lista.org/rewards?utm_source=gitbook\&utm_medium=article\&utm_campaign=lp-pools)直接在 Lista DAO 上质押他们的 PancakeSwap lisUSD/USDT 稳定 LP 代币。
2. **在外部平台上重新质押**：Lista DAO 将质押的 LP 代币重新质押到外部平台，如 PancakeSwap 农场，其中生成额外的奖励。对于 PancakeSwap，用户可以在[这里](https://pancakeswap.finance/farms)查看农场详情。
3. **实时 APR 显示**：用户可以实时跟踪 LP 发放 APR 和 LP 重新质押 APR。总 APR 是 Lista DAO 发放的奖励和外部平台农场奖励的组合。
4. 每个**星期三 0:00 UTC**，下面突出显示的每个池的周发放率将更新。这确保用户在质押 LP 代币或借用/质押 lisUSD 时获得实时代币发放。发放计划完全透明，用户可以每周跟踪他们的更新奖励。
