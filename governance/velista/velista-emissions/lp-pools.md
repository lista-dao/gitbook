# LP池

## 合格的LP池

以下是用户可以提供流动性以赚取LISTA奖励的所有合格池的表格。

| **协议** | **池名称**                  |
| -------- | -------------------------- |
| Lista DAO | lisUSD单独质押池 |
| Lista DAO | 借入lisUSD              |

## LP池奖励条件

为了有资格获得LP池类别下的LISTA奖励，用户必须在Lista DAO的[奖励](https://lista.org/rewards)部分质押他们的LP代币。

此外，不同类型的LP将必须满足某些条件。

## 内部LP

参与Lista DAO的DeFi生态系统的用户将有机会获得LISTA奖励。这些奖励作为社区激励措施中的LP池奖励分配。

### 1. lisUSD借款人

lisUSD借款人也有资格获得LISTA奖励。分配给每个用户的奖励与借入的lisUSD金额相比于所有借入的lisUSD的总价值锁定（TVL）成正比。

**规则和条件：**

1. 用户必须在Lista DAO的[平台](https://lista.org/cdp/loans)上借入lisUSD。
2. 奖励每秒发放一次，从lisUSD代币在Lista DAO平台上借出的那一刻开始计算。

### 2. lisUSD单独质押池

在Lista DAO的lisUSD单独质押池中质押他们的lisUSD的用户也将有资格获得LISTA奖励。分配给每个用户的奖励与质押的lisUSD金额相比于lisUSD单独质押池的总价值锁定（TVL）成正比。

**规则和条件：**

1. 用户必须在Lista DAO的[平台](https://lista.org/cdp/earn)上单独质押lisUSD。
2. 奖励每秒发放一次，从lisUSD代币在Lista DAO的lisUSD单独质押池上质押的那一刻开始计算。

## 外部LP

当用户在Lista DAO上质押他们的LP代币时，我们将这些代币重新质押到外部DeFi平台，如PancakeSwap和Thena Finance。通过这个过程，用户获得双重奖励——他们从Lista DAO获得LISTA发放以及从PancakeSwap（以CAKE奖励）和Thena（以THE奖励）获得的农场/LP奖励。

Lista DAO支持来自流行DeFi协议的多种LP代币，使用户能够有效地质押并赚取奖励。支持的LP代币包括以下内容：

| **协议**  | **池名称**                  |
| --------- | -------------------------- |
| Pancake Swap | lisUSD/USDT稳定池LP |
| Thena Finance | LISTA/USDT窄LP       |
|               | slisBNB/BNB相关LP  |
|               | lisUSD/FRAX（稳定）LP     |
|               | lisUSD/USDT（cl稳定）LP  |
|               | lisUSD/BNB ICHI LP         |

**规则和条件：**

1. **在Lista DAO上质押**：用户可以直接在Lista DAO上质押他们的PancakeSwap lisUSD/USDT稳定LP代币，通过[此链接](https://lista.org/rewards)操作。
2. **在外部平台上重新质押**：Lista DAO将质押的LP代币重新质押到外部平台，如PancakeSwap农场，其中生成额外的奖励。对于PancakeSwap，用户可以在[这里](https://pancakeswap.finance/farms)查看农场详情。
3. **实时APR显示**：用户可以实时跟踪LP发放APR和LP重新质押APR。总APR是Lista DAO发放的奖励和从外部平台的农场奖励获得的奖励的组合。
4. 每个**星期三0:00 UTC**，下面突出显示的每个池的周发放率将更新。这确保用户在质押LP代币或借用/质押lisUSD时获得实时的代币发放。发放计划完全透明，用户可以每周跟踪他们的更新奖励。
