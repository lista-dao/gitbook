# veLISTA

## 理解 veLISTA 和 veToken 模型 <a href="#id-91ff" id="id-91ff"></a>

要理解用户如何获得 veLISTA 以及 veLISTA 的用途，我们首先需要了解什么是 veToken 模型。veToken 模型由 Curve Finance 引入，是一种旨在使激励与鼓励代币持有者长期承诺的代币经济机制。在此模型中，用户将他们的代币（称为“veTokens”）锁定一段指定时间，在此期间，他们将获得治理权力和各种奖励。锁定时间越长，他们获得的 veTokens 就越多，这意味着更大的投票权和更高的奖励。

该系统激励用户承诺长期支持协议，促进稳定性并使他们的利益与平台的长期成功保持一致。因此，Lista 的代币锁定为 veLISTA 的做法受到 Curve Finance 的 veToken 模型的启发，并且功能类似。

## 走向去中心化 <a href="#d418" id="d418"></a>

在 Lista DAO 过渡到我们的 veToken 模型期间，治理权将逐渐移交给 veLISTA 持有者。用户将能够对几个参数进行投票。然而，这种转向完全去中心化的治理模型将是逐步的，而不是瞬间的。

为确保系统安全并防止恶意行为，核心团队最初将保留否决有害提案的权力，这与我们优先考虑 Lista DAO 最佳利益的使命相符。随着 Lista DAO 的成熟，更多的权力将逐渐转移给 DAO。我们的最终目标是完全过渡到一个治理拥有绝对权力的 DAO。

此外，一旦 veLISTA 启动，veLISTA 持有者将开始收到协议费用，进一步激励他们积极参与治理和 Lista DAO 的长期成功。

1. veLISTA 提前解锁费
