# veLISTA的计量表投票

## 什么是计量表？ <a href="#id-7f1f" id="id-7f1f"></a>

您可以将任何需要LISTA排放的产品（不包括veLISTA持有者排放）视为一系列的计量表。这包括我们奖励部分下的白名单LP、lisUSD单独质押和lisUSD借贷。

## 计量表投票 <a href="#a42c" id="a42c"></a>

计量表投票允许veLISTA持有者（那些已经锁定他们的LISTA代币以获得投票权的人）投票决定LISTA排放的方向。通过计量表投票，一个计量表累积的veLISTA越多，将会有更多的LISTA排放分配给相应的流动性池，这有效地决定了哪些池获得更多的支持和奖励。

每个时期（N）的投票决定了下一个时期（N+1）的LISTA排放量，这些变化只在当前时期结束后生效。

为了防止大持有者潜在的操纵，将为每个流动性池引入排放上限，确保奖励的公平和均衡分配。

## 计量表投票的工作原理 <a href="#e8d4" id="e8d4"></a>

1. **投票周期**：每个投票周期为一周。在第N周投出的票将影响第N+1周的LISTA代币分配。因此，如果您在第1周投票，您的投票将影响第2周的代币分配。
2. **投票过程**：在投票期间，veLISTA持有者可以将他们的投票权重的一定百分比分配给各个LP池。您可以分配最多100%的投票权，根据您的意愿跨不同池分配。一周结束时，将进行快照以根据每个池中的总票数确定分配。
3. **分配**：投票结束后，根据投票结果将LISTA代币存入每个池。例如，如果A池获得了总票数的20%，它将收到每周LISTA排放的20%。
4. **重置投票**：在每个新的投票周期开始时，所有投票权重都将重置为零，因此用户需要每周重新投票以影响即将到来的分配。

## 需要了解的关键事项 <a href="#id-8e60" id="id-8e60"></a>

* **动态分配**：每个池每周接收的LISTA数量将根据投票结果而变化。这意味着您的投票权可以直接影响特定LP的奖励分配。
* **每周承诺**：因为投票每周都会重置，鼓励积极参与以保持对分配的影响。记得定期回访并调整您的投票。
* **战略性投票**：考虑哪些池最有可能惠及生态系统或根据您的兴趣提供最佳回报。得票较高的池将获得更多代币，增加了参与这些LP的用户的潜在奖励。

在Lista DAO上的计量表投票使veLISTA持有者能够直接影响平台的代币分配。通过参与这一社区驱动的过程，您不仅增强了潜在的奖励，还有助于塑造Lista DAO上一个繁荣、平衡的生态系统。
