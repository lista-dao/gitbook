# Lista DAO：解锁 veLISTA 的实用性

Lista DAO 致力于扩展 veLISTA 的实用性，引入了一系列旨在为持有者提供长期利益的功能。这些更新旨在增强资本效率、提升用户参与度，并在生态系统内推动可持续增长。通过持有 veLISTA，用户将获得独家金融利益，改善他们的借贷和质押体验。

## lisUSD 返利机制

作为这些增强功能的首个，我们引入了 Borrow lisUSD 返利机制，这是一个直接减少 veLISTA 持有者借款成本的功能。

借款返利机制通过基于用户的 veLISTA 持有量提供 lisUSD 返利来奖励用户，使他们能够优化在 Lista DAO 生态系统内的借款体验。

随着 lisUSD 返利机制的推出，符合条件的 veLISTA 持有者现在可以同时从 lisUSD 借款返利和 LISTA 发放奖励中受益。

更多 veLISTA 实用性功能将很快公布

## 核心逻辑

借款返利百分比将根据用户持有的 veLISTA 数量计算，遵循以下函数：

F(x) = A\*ln(x) + B

其中：

* x = 用户持有的 veLISTA 数量
* ln = 自然对数函数
* f(x) = 借款返利百分比
* A 和 B 将根据市场条件进行调整，核心 Lista 团队拥有最终决定权。

函数遵循以下阈值：

* 如果 x ≤ 10k veLISTA，f(x) = 0（无返利）
* 如果 x ≥ 1000M veLISTA，f(x) ≤ 2.5%（最大返利上限）

目前函数设置如下：

&#x20; f(x) = 0.002171⋅ln(x) - 0.02

## 示例

### 示例 1：无返利（未达到阈值）

用户 A 借款 100,000 lisUSD，借款利率为 10%。\
用户 A 持有 10k veLISTA → 低于阈值，因此不适用返利。\
返利率：0%\
每周返利：0 lisUSD

### 示例 2：部分返利

用户 B 借款 100,000 lisUSD，借款利率为 10%。

用户 B 持有 1M veLISTA，应用函数：\
f(1,000,000) = 0.002171 × ln(1,000,000) - 0.02\
\= 0.00999347221 x 100%

\=0.99%

每周返利计算：(100,000 × 0.99%) / 365 × 7\
\= 18.986 lisUSD 每周

### 示例 3：达到最大返利上限

用户 C 借款 100,000 lisUSD，借款利率为 10%。\
用户 C 持有 10B veLISTA，超过了 1000M 上限。

返利率：2.5%（应用最大上限）。

每周返利计算：\
(100,000 × 2.5%) / 365 × 7\
\= 47.945 lisUSD 每周

## 用户流程

1. 借款 lisUSD：用户借用 lisUSD 并照常支付利息。
2. 返利计算：系统计算用户的 veLISTA 持有量并确定借款返利年化利率 (APR)。
3. 返利处理：根据上述函数，计算每小时的返利金额为 f(x) \* 借款 lisUSD 数量。
4. 返利周期：返利从第 N 周的星期三累积到第 N+1 周的星期三。
5. 领取返利：如果没有问题，用户可以在每周四领取他们的 lisUSD 返利。
