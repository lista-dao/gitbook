# LISTA通缩倡议：20%代币销毁（LIP-021）

随着[LIP-021](https://snapshot.box/#/s:listavote.eth/proposal/0xdb92580e04ca8d688404a689a99579294715bd9ab858cc9b7ba637599720a6ff)的成功实施，Lista DAO已永久销毁了2亿LISTA代币，占最大供应量的20%。

这一战略举措将总最大供应量从10亿减少到8亿LISTA，立即产生了通缩影响，以增强协议的长期价值和可持续性。

### LISTA代币销毁

* **销毁数量**：2亿LISTA（原始最大供应量的20%）。
* **新的最大供应量**：8亿LISTA。
* **执行方法**：通过将代币转移到黑洞地址来永久从流通中移除，确保它们永远无法恢复或使用。

### **交易详情**

* 黑洞地址：[https://bscscan.com/token/0xfceb31a79f71ac9cbdcf853519c1b12d379edc46?a=0x000000000000000000000000000000000000dead](https://bscscan.com/token/0xfceb31a79f71ac9cbdcf853519c1b12d379edc46?a=0x000000000000000000000000000000000000dead)

> _注意\* 目前，总冻结的LISTA约为204.41M LISTA，包括一次性燃烧（LIP 021，200M）永久移除以减少供应并增加稀缺性，以及基于_[_LIP 016_](permanent-locking-of-lista-lip-016.md)_的之前累积的协议冻结（4.41M）。这代表了加强代币长期价值和支持DAO生态系统的战略步骤。_
