# LISTA 分配

根据 [LIP-021](https://snapshot.org/#/s:listavote.eth/proposal/0xdb92580e04ca8d688404a689a99579294715bd9ab858cc9b7ba637599720a6ff)，2亿 LISTA 被永久锁定，LISTA 的总供应量从10亿减少到8亿。在此提案之前和之后，所有 LISTA 代币的分配情况如下：

| 分配对象               | 发行时的百分比 | LIP-021后的百分比 |
| ---------------------- | -------------- | ---------------- |
| 永久锁定               | 0.00%          | 20.00%           |
| 币安 Megadrop          | 10.00%         | 10.00%           |
| 空投                   | 10.00%         | 10.00%           |
| 投资者与顾问           | 19.00%         | 17.00%           |
| 团队                   | 3.50%          | 2.50%            |
| 社区                   | 40.00%         | 26.00%           |
| 储备金（DAO 财库）     | 8.00%          | 5.00%            |
| 生态系统               | 9.50%          | 9.50%            |

### 币安 Megadrop

向币安 Megadrop 分配的全部代币供应量将在发行时解锁。

### 空投

发行时，分配给 Lista DAO 空投的代币中的85%，即总代币供应量的8.5%，将被解锁。

剩余的15%，即 LISTA 总代币供应量的1.5%，将在2025年第二季度解锁。

### 投资者与顾问

分配给投资者与顾问的总代币供应量中的1%将在发行时解锁。

其余的18%将被锁定一年，直到2025年6月。从2025年6月开始，这部分将按季度线性解锁，持续两年，直到2027年3月。

### 团队

分配给 Lista DAO 团队的全部代币供应量（3.50%）将被锁定一年，直到2025年6月，首次解锁将在2025年6月进行。

从2025年6月开始，分配给 Lista DAO 团队的3.50%代币供应量将按季度（每3个月）线性解锁，持续5年，直到2029年3月。

### 社区

在发行时，分配给社区激励的代币供应量将用于推动各种 DeFi 活动的代币排放，如 Lista DAO 的 LP 奖励、lisUSD 单币质押池等。

LISTA 代币向这些社区激励的排放将从7月开始，并将按每年1/√2的递减速率持续排放，直到2044年6月。

社区激励的代币排放将遵循以下公式：![](<../../.gitbook/assets/image (26).png>)

**n**：当 n = 1 时，代表第一年的代币排放。在最后一年，n=20

**r**：r 是几何级数的公比：1/√2

**S**：S 是几何级数的和：即代币排放的40%，相当于4亿 LISTA

由于 Lista DAO 仍处于过渡阶段，当前的排放率设定为 **每周540,000 LISTA**，低于预期的每月9,772,651 LISTA 的排放率。

#### Lista DAO 过渡阶段

在 Lista DAO 向完全去中心化自治组织（DAO）过渡期间，原计划的 Lista DAO 代币排放（第一年每月9,772,651 LISTA）将被分成两个相等的部分，每部分为4,886,325.5 LISTA。

前半部分：第一年每月4,886,325.5 LISTA 的第一半代币排放将根据当前需求每月动态调整。在排放这部分代币之前，社区将被适当通知。

后半部分：第一年每月4,886,325.5 LISTA 的第二半代币排放将用于社区激励，但不会立即排放。排放量将根据 Lista 平台部署的进展逐渐增加。排放增加将被限制在每周1,221,581 LISTA，换句话说，长期来看是每月4,886,325.5 LISTA。

目前，LISTA 的排放率设定为 **每周540,000 LISTA**。所有剩余未发行的 LISTA 代币将继续分配给社区激励，未来的排放率决定将通过 DAO 投票进行。随着我们完成向 DAO 的过渡，调整 LISTA 当前排放参数的权力也将逐渐转移给 veLISTA 持有者。

### DAO 财库

分配给 Lista DAO 财库的全部代币供应量（8.00%）将被锁定三个月，直到2024年9月，首次解锁将在2024年9月进行。

从2024年9月开始，分配给 Lista DAO 财库的8.00%代币供应量将按季度（每3个月）线性解锁，持续4年，直到2028年6月。

### 生态系统

分配给生态系统激励的总代币供应量中的3.50%将在发行时解锁。
