# 概览

Lista DAO 是一个去中心化自治组织，是一个BNB的借贷协议、流动性质押解决方案以及BNB Chain上的抵押债务位置（CDP）协议。Lista DAO 还得到了币安实验室（现YZi Labs）的支持，战略投资额为1000万美元。

自我们启动以来，Lista DAO 已经达成了几个里程碑，包括我们的治理代币LISTA在币安的官方TGE（2024年6月20日），关键更新如我们的AMO，ListaPie的启动，以及我们的Lista创新区的推出。我们与不同协议的战略合作伙伴关系也扩展了我们的生态系统，并为我们的用户带来了额外的价值。通过坚持我们的核心价值观并持续创新，Lista DAO 力求成为DeFi领域的领导者，为用户提供安全、高效和赋能的金融解决方案。

## 使命

我们的使命是通过解决流动性供应中的关键挑战并为BNB提供一流的流动性质押解决方案，推动DeFi的发展。通过如流动性质押和CDP机制等强大的产品，Lista DAO 旨在为用户提供高效和安全的金融工具。

## 愿景

我们对Lista DAO的长远愿景是将其建立为DeFi生态系统的基石，促进我们产品的广泛采用。我们优先考虑持续发展，不断创新并引入新功能和服务以满足用户的不断变化的需求。透明度是我们运营的基础，确保所有流程和决策对社区透明和可访问。Lista DAO 坚持最高的安全标准，实施全面的措施并进行定期审计，以维护我们平台和用户资金的完整性和安全性。有关更详细的信息，您可以在[此处](security/audit-reports.md)查看我们的审计报告。

## 核心功能

### 借贷

2025年4月推出的 [Lista Lending](https://lista.org/lending/borrow?utm_source=medium\&utm_medium=text\&utm_campaign=year_report) 在六个月内成为BNB Chain上最大的借贷市场之一。Lista Lending 提供了数十个市场和借贷金库，允许第三方通过其智能合约创建无需许可的金库，同时提供一流的金库以最大化您在BNB Chain上的资产效率。Lista Lending 的最新章节是 Lista Credit。

### 智能借贷与交换

智能借贷是Lista的下一代借贷解决方案，其中抵押品存入Lista的DEX，Smart Swap的流动性池。这开辟了一条全新的收入来源——交易费。如果您将BNB/slisBNB存入智能借贷作为流动性供应，您将能够借出USD1或BNB，同时从Smart Swap上的BNB/slisBNB交易活动中赚取交易费。

### 抵押债务位置（CDP）

仿照MakerDAO，我们的CDP系统允许用户锁定各种抵押品并兑换稳定币（lisUSD），为资产杠杆提供安全的方式。

### 流动性质押解决方案

受Lido DAO启发，我们的流动性质押机制允许用户质押他们的BNB，赚取奖励，同时保持流动性。这增强了DeFi生态系统内的整体流动性，同时确保了用户的灵活性和可访问性。
