# 概览

Lista DAO是一个去中心化自治组织，主要提供BNB的流动性质押解决方案，并且是一个抵押债务位置（CDP）协议，用户可以借用我们的稳定币lisUSD，对抗多种抵押品选项。Lista DAO还得到了币安实验室的支持，策略性投资了1000万美元。

自我们启动以来，Lista DAO已经达成了几个里程碑，包括我们的治理代币LISTA在币安（2024年6月20日）的官方TGE，关键更新如我们的AMO，ListaPie的启动，以及我们的Lista创新区的推出。我们与不同协议的战略合作也扩展了我们的生态系统，并为我们的用户带来了额外的价值。通过坚持我们的核心价值观并持续创新，Lista DAO力求在DeFi领域成为领导者，为用户提供安全、高效和赋能的金融解决方案。

## 使命

我们的使命是通过解决流动性供应中的关键挑战，并为BNB提供一流的流动性质押解决方案，推动DeFi的发展。通过强大的产品如流动性质押和CDP机制，Lista DAO旨在为用户提供高效和安全的金融工具。

## 愿景

我们对Lista DAO的长期愿景是将其建立为DeFi生态系统的基石，促进我们产品的广泛采用。我们优先考虑持续发展，不断创新并引入新的功能和服务以满足用户的不断变化的需求。透明性是我们运营的基础，确保所有过程和决策对社区透明和可访问。Lista DAO维持最高的安全标准，实施全面的措施并进行定期审计，以保持我们平台和用户资金的完整性和安全。更多详细信息，您可以在[此处](security/audit-reports.md)查看我们的审计报告。

## 核心功能

### 抵押债务位置（CDP）

仿照MakerDAO，我们的CDP系统允许用户锁定多种抵押品并针对其铸造稳定币（lisUSD），提供一种安全的资产杠杆方式。

### 流动性质押解决方案

受Lido DAO启发，我们的流动性质押机制允许用户质押他们的BNB，赚取奖励，同时保持流动性。这增强了DeFi生态系统内的整体流动性，同时确保了用户的灵活性和可访问性。
