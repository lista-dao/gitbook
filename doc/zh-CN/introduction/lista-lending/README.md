# Lista Lending

Lista Lending 是一个完全去中心化且无需许可的 P2P 借贷协议，专为 BNB Chain 设计，突破传统大型借贷池的限制，培育更包容和弹性的生态系统。

Lista Lending 的核心围绕一个基于保管库的系统，汇集流动性并根据供需动态分配到不同的借贷和抵押配对，我们称之为市场。

## 核心组件

### 抵押化

借款人必须将支持的加密资产作为抵押品存入其选择的市场，以解锁与抵押品价值（LTV 比率）成比例的借款能力。

### 清算贷款价值比 (LLTV)

当借入资产的价值达到抵押品价值的一定百分比（例如 90%）时，抵押品被视为不足，借款人面临被清算的风险。

### 借款

借款人指定他们希望从市场借入的资产数量，并提供所需的抵押品。

### 利率

借款人按照商定的利率模型支付利息，利息随时间累积，并在还款时支付。

### 还款

借款人可以随时归还贷款本金和利息，并在交易确认后取回抵押品。

### 清算机制

如果由于市场波动或利息累积，借入资产的价值超过市场设定的清算贷款价值比（LLTV），则借款人的位置可能会部分或完全被清算。

### 出借

出借人（供应商）将一定数量的资产存入保管库或市场，相应的保管库/市场将这些资产借给借款人并获得利息收益。

### 提款

出借人可以随时（取决于市场流动性）提取存入的资产和利息。
