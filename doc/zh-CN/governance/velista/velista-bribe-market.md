# veLISTA 贿赂市场

## 什么是贿赂市场？ <a href="#id-0384" id="id-0384"></a>

贿赂市场是一项新功能，使项目和用户能够向 **veLISTA** 持有者提供奖励 —— 通常称为“贿赂” —— 以换取他们对特定 LP 池的投票。通过激励投票者，项目可以将更多的 LISTA 发放向他们偏好的池，促进更大的流动性和参与度。

贿赂通过 **投票激励模块** 透明地分发，确保 veLISTA 投票者获得公平和去中心化的奖励机制。

## 为什么这很重要？ <a href="#id-4f34" id="id-4f34"></a>

## 对于 veLISTA 投票者 <a href="#id-342e" id="id-342e"></a>

* **赚取更多奖励**：投票者现在可以通过参与贿赂激励的投票获得额外的奖励，如 LISTA 或其他代币。
* **增加参与度**：贿赂市场为治理增加了一个令人兴奋的策略层，使您的投票更具影响力和回报。

## 对于项目 <a href="#id-8489" id="id-8489"></a>

* **提升流动性**：项目可以通过提供贿赂来激励其池中的流动性，吸引更多的发放并增加用户参与。
* **直接发放**：贿赂为项目提供了一个强大的工具，以将发放引导至对其生态系统最重要的池。

## 对于生态系统 <a href="#fd94" id="fd94"></a>

* **加强治理**：引入贿赂市场在治理中创造了更积极的参与，确保 Lista DAO 生态系统更健康、更去中心化。
* **增强透明度**：所有贿赂奖励及其分配都被追踪和显示，增进社区内的信任和开放性。

## 它是如何工作的？ <a href="#id-50c7" id="id-50c7"></a>

## 对于 veLISTA 投票者 <a href="#id-67fd" id="id-67fd"></a>

1. **对 LP 池投票**：使用您的 veLISTA 为您想要支持的 LP 池投票。
2. **赚取贿赂奖励**：如果一个池子被贿赂激励，您将根据您的投票权重获得奖励的一部分。

_示例：_

* 一个项目为 Pool A 的投票提供 1000 LISTA 作为贿赂。如果您持有 Pool A 总投票的 10%，在投票期结束后，您将获得 **100 LISTA** 作为奖励。

## 对于提供贿赂的项目或用户 <a href="#id-0116" id="id-0116"></a>

1. **选择一个 LP 池**：选择您想要激励的池。
2. **存入奖励**：向贿赂市场添加批准的资产（例如 LISTA, USDC）作为奖励。
3. **设置 votingPeriod**：选择您想要激励的投票期 —— 当前周、下一周或在多个周中分配奖励。
4. **奖励分配**：一旦投票期结束，贿赂将自动分配给 veLISTA 投票者。

## 贿赂市场的关键特性 <a href="#id-6204" id="id-6204"></a>

1. **透明和去中心化**：所有贿赂都公开可见，其分配完全基于投票权重自动进行。
2. **白名单资产**：只有批准的代币可以用于贿赂，确保奖励有价值且可信。
3. **灵活的周期**：贿赂可以为单一投票期分配，或在多个周中均匀分配。

## 对于 veLISTA 用户的好处 <a href="#a073" id="a073"></a>

* **更高的 APR**：贿赂为治理增加了额外的奖励层，显著提升投票者的 APR。
* **战略性投票**：投票者可以根据个人偏好、奖励或长期策略选择池。
* **复合激励**：用户可以将其贿赂奖励再投资于 veLISTA，增加他们的投票权和未来收益。
