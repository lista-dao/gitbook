# Lista 借贷风险基金

## Lista DAO 风险基金

**风险基金地址：0x618579671f4B5a96Ff6Ac3Fb66224df39Ce9d325**

## 概览

Lista DAO 风险基金是一个专门的储备金，旨在增强 Lista 借贷生态系统的安全性和韧性。

通过将 Lista DAO 自行创建的 Vaults 下产生的 50% 收入分配给该基金，该基金作为财务缓冲，保护用户免受由于 Lista 自营 Vaults 中的漏洞或不可预见事件导致的潜在损失。

_**注意：不由 Lista DAO 管理/创建的 Vaults 不包括在此福利中**_

## 基金的目的

Lista DAO 风险基金的主要目标是通过提供一种机制来覆盖安全漏洞或其他影响 Lista DAO 借贷 Vaults 的关键事件中的财务损失，从而保护用户。具体来说，基金用于：

* **赔偿用户损失**：在确认的漏洞或导致用户在 Lista DAO 的 Vaults 中损失资金的黑天鹅事件的情况下，将使用该基金来偿还受影响的用户，确保公平及时的赔偿。\
  &#xNAN;_&#x54;包括清算损失、坏账以及可能发生在 Lista DAO 创建的 Vaults 中的任何其他潜在风险。_
* **增强协议韧性**：通过维护一个强大的储备金，该基金减轻系统性风险，增强 Lista 借贷平台的整体稳定性。

## 资金机制

风险基金通过从 Lista 自营 Vaults 的持续收入分配来维持：

* 来源：Lista 自营 Vaults 收取的管理费的一半被直接用于风险基金。
* 费用结构：Vaults 收取的总管理费为 10%，其中一半，即 5%，分配给风险基金。
* 频率：随着管理费的收取，持续向基金贡献，确保储备的稳定增长。

这种分配策略平衡了协议的盈利性与用户保护，利用 Lista 的运营收入构建一个可持续的安全网。

## 管理与治理

风险基金由 Lista DAO 基金会集中管理，以确保透明度、效率和问责制：

* 监督：Lista DAO 核心团队将负责基金的管理，包括其分配和部署。
* 使用条件：仅在确认的漏洞或导致可验证用户损失的事件中激活基金。建立明确的基金拨付标准和流程，以确保公平和公正的赔偿。

## 运营细节

* 范围：基金仅覆盖与 Lista 自营 Vaults 中的漏洞或安全漏洞相关的损失。它不扩展到市场波动、用户错误或外部协议风险的损失。
* 拨款流程：在确认漏洞后，Lista DAO 核心团队将评估损失范围，验证受影响的用户，并从基金中执行还款。该过程优先考虑速度和公平性，并向社区进行明确的沟通。
* 基金安全：基金的资产保存在由 Lista DAO 基金会管理的安全的多签名钱包中，严格的访问控制以防止未经授权的使用。
