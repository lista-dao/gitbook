# Lista Lending Earn

{% embed url="https://www.loom.com/share/bd0a7948601945c191c7792fab0cbbb9?sid=c78dc44f-9565-4ed0-ab37-29cf191d63e0" %}

1. 访问 Lista Lending 的赚取页面 [这里](https://lista.org/lending?tab=earn&utm_source=gitbook&utm_medium=article&utm_campaign=lista-lending-earn)。

<figure><img src="../../.gitbook/assets/image (62).png" alt=""><figcaption></figcaption></figure>

2. 点击经典抵押品。 （要参与 Alpha 区域，请阅读 [这里](../../introduction/lista-lending/vaults/lista-alpha-zone.md)）
3. 在左上角筛选策展人和存款资产类型，或搜索您正在寻找的金库名称。

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure>

4. 选择您正在寻找的金库。在此示例中，我们将使用 Lista USD1 Vault 作为示例。

<figure><img src="../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>

5. 输入金额并点击“存款”。确认交易，完成操作。

<figure><img src="../../.gitbook/assets/image (65).png" alt=""><figcaption></figcaption></figure>

6. 在您的仪表板中检查并监控您的位置 [这里](https://lista.org/dashboard?utm_source=gitbook&utm_medium=article&utm_campaign=lista-lending-earn)。

<figure><img src="../../.gitbook/assets/image (66).png" alt=""><figcaption></figcaption></figure>
