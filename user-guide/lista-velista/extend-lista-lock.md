# 延长 LISTA 锁定

用户可以随时选择延长他们的 LISTA 锁定。

## 如何操作：

{% embed url="https://www.loom.com/share/5a8ab7f67efb4633a608dd4d3013fdaf?sid=26cf5c4c-18bf-427d-a964-699d2246a034" %}

1. 要延长您的 LISTA 代币锁定，请访问 Lista DAO 的锁定页面[此处](https://lista.org/lock?utm_source=gitbook&utm_medium=article&utm_campaign=extend-lista-lock)，并将您的钱包连接到 Lista DAO 的 DApp。
2. 在“您的锁定”部分，点击“延长”

<figure><img src="../../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

3. 根据您的偏好调整并延长锁定时间，点击“延长”按钮，并签署交易。

<figure><img src="../../.gitbook/assets/image (9).png" alt=""><figcaption></figcaption></figure>

4. 一旦“扩展”交易完成，您就完成了！您的新解锁日期将显示在您的锁定仪表板上。
