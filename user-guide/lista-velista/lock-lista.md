# 锁定 LISTA

{% embed url="https://www.loom.com/share/01f63aaceea44353994659b8f219ba47?sid=4d13d604-07b9-4d8d-886a-7095d7a0ab55" %}

1. 访问网站[这里](https://lista.org?utm_source=gitbook&utm_medium=article&utm_campaign=lock-lista)，并将您的钱包连接到 Lista DAO 的 DApp。
2. 在网页顶部，点击“veLISTA”，然后点击“锁定 veLISTA”。

<figure><img src="../../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

2. 输入您想要锁定到 veLISTA 的 LISTA 代币数量。\
   注意\* LISTA 的转换率如下：

   1 LISTA（锁定 52 周）= 52 veLISTA\
   1 LISTA（锁定 20 周）= 20 veLISTA\
   1 LISTA（锁定 1 周）= 1 veLISTA

<figure><img src="../../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

2. 自动锁定功能默认开启。这意味着如果用户不关闭此功能，LISTA 将会每周自动重新锁定。\
   \
   如果关闭此功能，用户的 veLISTA 将会每周（每周三 UTC+0）减少，直到第 0 周，届时锁定的 LISTA 将被解锁，用户可以随时领取他们的 LISTA 代币。

<div align="left"><figure><img src="https://lh7-us.googleusercontent.com/docsz/AD_4nXesdGoGxsqK_6RV8Q0fu0YEkXpESCMoQbYqTw9Jtz1dKYMNXgo7oCDmuJ0scSbzTXQdhs0RS34iQBQiVdrUF3mEAvQI5iuyFvaMN0y7JjasvpPi3LBZWLBzI9kCFfe2c0ZNCfUZmMF5tGTzwavNAL5AXfiA?key=I9PhIveewXW6geOxMvM3tQ" alt=""><figcaption></figcaption></figure></div>

5. 点击“批准”，并签署交易。
6. 完成“批准”交易后，点击“锁定”，并签署交易。
7. 完成“锁定”交易后，您就完成了！您可以在同一页面的“您的锁定”下查看您的 veLISTA 持有情况。
