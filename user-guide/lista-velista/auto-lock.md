# 自动锁定

用户可以随时选择开启“自动锁定”功能，以避免每周手动重新锁定他们的LISTA代币的麻烦。用户还可以选择他们希望自动锁定LISTA代币的周数。

以下是操作方法：

{% embed url="https://www.loom.com/share/210bd3a6377e445fa1c3d85701af2dce?sid=9c33d62a-b348-42ef-b0ca-66201ded3754" %}

1. 访问Lista DAO的锁定页面[这里](https://lista.org/lock?utm_source=gitbook&utm_medium=article&utm_campaign=auto-lock)，并将您的钱包连接到Lista DAO的DApp。
2. 在“您的锁定”部分，只需点击右上角的“自动锁定”功能即可开启。

<figure><img src="../../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

3. 用户可以选择将他们的LISTA代币自动锁定任意时间长度，从1周到52周不等。在这个例子中，我们选择将LISTA代币永久性地自动锁定16周。一旦您确认了自动锁定周期，在第2步开启自动锁定功能后，将会出现一个弹窗。点击“确认”，并签署交易。

<figure><img src="../../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

4. 一旦“启用自动锁定”交易完成，您就完成了！
