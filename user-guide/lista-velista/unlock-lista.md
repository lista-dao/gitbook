# 解锁 LISTA

用户可以随时通过支付提前解锁费用来解锁他们的 LISTA 代币。如果他们想避免支付提前解锁费用，他们必须等到锁定期结束。

请注意，如果用户想避免支付任何提前解锁费用，他们必须关闭自动锁定功能，并等待他们的 LISTA 代币完全解锁。

以下是操作方法：

{% embed url="https://www.loom.com/share/e962c9e1bc8d49b68470c388104ab048?sid=301a6a71-4264-44f2-9b00-ad053c22c3eb" %}

1. 访问 Lista DAO 的锁定页面[这里](https://lista.org/lock?utm_source=gitbook&utm_medium=article&utm_campaign=unlock-lista)，并将您的钱包连接到 Lista DAO 的 DApp。
2. 在“您的锁定”下，点击“解锁”。

<div align="left"><figure><img src="../../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure></div>

3. 将出现一个弹出窗口，显示您需要支付的提前解锁费用的确切金额，以解锁您的锁定 LISTA 代币。\
   \
   如果您在申领代币之前已完成锁定期，提前解锁费用将为 0，您无需支付任何额外费用。确认提前解锁费用金额后，点击“解锁”，并签署交易。

<div align="left"><figure><img src="../../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure></div>

4. 交易完成后，您将收到您的 LISTA 代币（扣除任何提前解锁费用），操作完成！
