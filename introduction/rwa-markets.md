# 真实世界资产市场

真实世界资产（RWA）市场提供了加密资产购买代币化的真实世界资产，如短期美国国债和抵押贷款义务，这些RWA提供了可预测的利率和不同的风险水平。

### 工作原理

当下达购买订单时，Lista将铸造与购买金额相同价值的RWA代币，并将其分发给买家。这些代币具有收益性，因此其价值会随时间增长。每周二，Lista将计算过去一周的利息，并将其计入每个RWA代币的价格中。请注意，Lista将收取所赚取利息的小部分（目前为5%）作为费用。

这些RWA代币也可以被销毁以赎回其价值并提取流动性。底层工具 - 美国国债、公司债务抵押等 - 只能在工作日出售，因此您的赎回可能需要5至14天时间才能完成。请注意，Lista将收取0.1%的赎回费。

所有RWA代币的购买和赎回都通过Janus Henderson Anemoy Treasury Fund（基金）处理，该基金是一个由Centrifuge支持的代币化英属维尔京群岛（BVI）专业基金。该基金由英属维尔京群岛金融服务委员会（FSC）授权，并向非美国投资者开放。

### Dowsure RWA代币

#### 产品详情

| 参数                  | 详情                    |
| ---------------------- | ----------------------- |
| 目标APY                | 约10%                   |
| 锁定期                 | 90天                    |
| 最低存款               | 100美元                 |
| 金库容量               | 300万美元（扩展至1500万美元）|
| 订阅窗口               | 第-7天至第0天           |
| 历史坏账率             | < 1.5%（5年平均）       |
| 自动续订               | 是（第83天可选择退出）  |

Dowsure RWA代币由以下三种底层资产支持：

1\. Payoneer Advance 这代表亚马逊跨境卖家的预结算应收账款。像亚马逊这样的平台通常需要14天以上时间结算，而Dowsure提前为商家垫付资金。周期范围从14天到28天，流动性通常较高。

2\. Dowsure的即买即付供应商发票融资，用于大额购买（50万美元以上）。商家提前收到货物，而Dowsure则弥补支付差额。周期范围从20天到30天。客户主要是贷款规模较大的商家（通常超过50万美元）。

3\. LOC信用额度 Dowsure的旗舰产品，有5年的跟踪记录。该产品为亚马逊、eBay和沃尔玛上年收入超过1000万美元的成熟电商商家提供信用额度。由汇丰银行、渣打银行、恒隆银行、招商银行、建设银行以及多个基金提供资金。5年内坏账率保持在1.5%以下。

### 安全与风险控制

* 3PAR自动还款：3PAR是Dowsure与亚马逊、沃尔玛、eBay和Payoneer共同开发的融资平台。经过商家授权后，平台销售收入会自动转至Dowsure，确保在源头稳定还款。
* DAL双重锁定：一种专利资金控制系统，锁定商家支付账户，验证实际业务控制者，并在违约事件中启用资金冻结或强制扣款。
* 符合KYC + AML：对所有借款人完成全面的客户身份识别和反洗钱流程。

提前退出：在第90天之前，您可以随时提取90%的本金（根据已投天数收取收益罚款）。剩余的10%将保留至第90天并在坏账审计后释放。

坏账政策：如果坏账率超过4%的APY，超出部分将从用户收到的收益中扣除。5年历史平均值低于1.5%。
