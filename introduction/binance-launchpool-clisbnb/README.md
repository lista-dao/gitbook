# Binance Launchpool: slisBNBx

slisBNBx 是用户在 Lista DAO 平台上将 BNB 存入抵押债务仓位（CDP）后获得的“凭证”代币。它旨在以 1:1 的比例代表用户的 BNB，确保每个 slisBNBx 直接与 CDP 中锁定的 BNB 数量相连。这意味着用户每存入 1 BNB，就会收到 1 slisBNBx。

该产品专为希望利用其 BNB 存款参与 Lista DAO 上其他 DeFi 活动的用户量身定制。

## slisBNBx 能做什么？ <a href="#id-9a96" id="id-9a96"></a>

slisBNBx 的主要功能是允许用户在 ListaDAO 上以其 BNB 抵押借入 lisUSD，同时仍能使用 slisBNBx 参与 Binance launchpool。持有 slisBNBx 的用户可以在其 **Binance Web3 MPC 钱包** 中利用 BNB 参加独家代币发行活动，赚取新代币，而无需关闭在 Lista DAO 上的债务仓位。

这种借贷与参与 Binance Launchpool 的强大组合为用户的 BNB 持有提供了更多的实用性和灵活性，所有这些都通过一个无缝的产品实现。

## slisBNBx 的关键特性 <a href="#id-8293" id="id-8293"></a>

slisBNBx 具有几个区别于其他代币的特点，为用户提供了专注的实用性，同时保持对其资产的控制：

### 1. 与 BNB 1:1 比例 <a href="#d449" id="d449"></a>

当用户将 BNB 存入 Lista DAO 的 CDP 时，他们以 1:1 的比例获得 slisBNBx。这种直接关系确保每存入一个 BNB，就会创建一个 slisBNBx，保持存款与凭证之间的无缝链接。

### 2. 不可转移的代币 <a href="#e763" id="e763"></a>

slisBNBx 的一个重要特性是它是不可转移的。一旦发行，slisBNBx 不能在钱包或用户之间移动。它仅与用户的 BNB 存款相关联，并且仅存在于 Lista DAO 平台的上下文中，确保了其安全性和专注的使用。

### 3. 提取时自动销毁 <a href="#f3b4" id="f3b4"></a>

当用户从 Lista DAO 的 CDP 提取其 BNB 或相应的 slisBNB 时，slisBNBx 会自动被销毁或摧毁。这种机制确保 slisBNBx 始终准确地代表用户仓位中的 BNB 数量。一旦提取了底层的 BNB，相关的 slisBNBx 就会从系统中移除，以防止价值不匹配。

### 4. 向另一个地址铸造 slisBNBx <a href="#id-7699" id="id-7699"></a>

slisBNBx 为用户提供了一个独特的功能：在将 BNB 存入 CDP 时，他们可以选择将 slisBNBx 铸造到另一个地址。然而，一旦选择了这个地址并发送了 slisBNBx，它就不再能转移到任何其他钱包。这个功能提供了更大的灵活性，特别是对于可能希望委托与另一个钱包相关的某些操作的用户，例如积累星尘点数。

然而，这种灵活性带来了一些关键限制：

- 存款时只能指定一个地址来接收 slisBNBx。
- 如果提取了底层的 BNB 或 slisBNB，铸造到次级地址的任何 slisBNBx 都会自动销毁。
- 作为一种新产品，slisBNBx 目前仍功能有限。截至目前，它仅作为用户存款 BNB 的收据，并且在 Lista DAO 平台上无法执行任何操作。
