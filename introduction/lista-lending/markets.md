# 市场

市场是一个独立的借贷池，将一个抵押资产与一个贷款资产配对（例如：USDT/BNB）。每个市场独立运作，隔离风险以防止溢出，并在启动后保持不变。每个金库可以拥有多个市场。创建市场是无需权限的。

关键特性包括以下几点：

1. 每个市场一个抵押资产，一个贷款资产
2. 贷款参数是不可变的
3. 每个市场相互独立运作
4. 无需权限：新市场创建不需要治理投票 - 更多信息
5. 透明规则：借贷条件清晰明确

#### 无需权限的市场创建

Lista Lending 的一个突出特点是其无需权限的市场创建，使用户能够部署由关键参数定义的独立借贷市场。这种方法与传统借贷平台不同，并通过赋予用户自定义市场的能力而改变了模式，无需中央监管。

与传统借贷协议相比，这些协议：

1. 需要治理批准来列出资产或调整参数。
2. 将资产聚集到一个共享池中，将风险分布到整个协议中，

Lista Lending 允许市场独立运作，参数在创建时设置，并通过其可升级的金库系统灵活管理。

#### 市场创建机制

在 Lista Lending 中，用户可以通过指定以下内容来创建市场：

* 贷款资产：要借出或借入的资产（例如，lisUSD）。
* 抵押资产：用于保障贷款的资产（例如，slisBNB）。
* 清算贷款价值比（LLTV）：80%
* 预言机：Chainlink、Binance Oracle、Redstone、API3
* 利率模型：协议的自适应利率模型，根据使用情况动态调整。
