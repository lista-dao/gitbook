# Markets

A Market is an isolated lending pool that pairs one collateral asset with one loan asset (eg; USDT/BNB). Each market operates independently, isolating risks to prevent spillover, and remains immutable after launch. Each vault can have multiple markets. Creating a market is permissionless.&#x20;

Key features includes the following:

1. One collateral asset, one loan asset per market
2. Loan parameters are immutable
3. Each market operates independently from one another
4. Anyone can create a new market, and governance approval is not required
