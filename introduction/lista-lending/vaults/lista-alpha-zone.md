# Lista Alpha Zone

Lista Alpha Zone Vaults 是 Lista Lending 内的专门借贷池，专为 Binance Alpha 代币设计。这些金库允许白名单用户：

* **存入 Alpha 代币作为抵押品**：将您的 Alpha 代币存入金库，解锁流动性，同时保留其潜在上升空间的暴露。
* **以抵押品借款**：使用您存入的 Alpha 代币借入资产，如 lisUSD 或其他支持的代币，实现对冲或扩大您的头寸等策略。
* **灵活提款**：随时提取您的抵押品。

## 如何参与 Alpha Zone <a href="#c85e" id="c85e"></a>

目前，参与 Alpha Zone Vaults 是仅限白名单的，以确保控制推出，管理风险并优先考虑用户安全。以下是您可以参与的方式：

1. **申请白名单访问**：

* 访问我们的白名单申请表格进行申请：[白名单申请](https://docs.google.com/forms/d/1UdSrlQlZlj2b9wJschIYiw11A352PmbDKoQFrR3NONc/edit)。
* 只有白名单地址可以存入金库或从市场借款，存款和借款受到限制以增加安全性。

**2. 存款和借款（一旦被白名单批准）**：

* 批准后，导航至[ lista.org](https://lista.org?utm_source=gitbook&utm_medium=article&utm_campaign=lista-alpha-zone)下的“Lending”部分的 Alpha Zone Vaults。
* 将您的 Alpha 代币存入金库。
* 使用您的抵押品借入资产，如 lisUSD，执行您的 DeFi 策略，还款和提款不受限制，以提供灵活性。

## 为什么选择 Lista Lending Alpha Zone？ <a href="#id-8af8" id="id-8af8"></a>

Alpha Zone Vaults 将围绕 Binance Alpha 代币的炒作转化为可行的实用性，提供一种安全的借款、对冲和扩大您头寸的方式。

通过提供一个受控环境（白名单），我们确保用户可以自信地参与这些新兴资产，同时降低风险。
