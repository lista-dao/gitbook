# D3M - 直接存款模块

## 什么是 D3M？ <a href="#e483" id="e483"></a>

D3M，即直接存款模块，是一种机制，它能够直接铸造稳定币 — 在本例中为 lisUSD — 并将其存入 DeFi 平台以增强流动性和优化市场利率。与传统的以抵押资产为基础的铸币不同，D3M 利用基于信用的铸币，允许更灵活的供应管理，并为生态系统增长创造新机会。

Lista DAO 的 D3M 将直接将 lisUSD 注入 DeFi 借贷平台，为 lisUSD 创造新的使用案例，同时为用户提供稳定的借贷环境。

## 铸造的 lisUSD 的限制使用 <a href="#id-66a3" id="id-66a3"></a>

为确保系统的完整性，通过 D3M 铸造的所有 lisUSD 将专门分配给 Aave 和 Venus 等选定平台上的流动性池。这确保了资金直接有助于 lisUSD 的 DeFi 应用，无风险滥用。

通过增强这些平台上的流动性，lisUSD 获得更多的实用性，促进用户之间更大的采用和需求。

## 市场利率优化 <a href="#e925" id="e925"></a>

D3M 引入了一个先进的机制来控制市场利率，通过在借贷平台上动态调整 lisUSD 供应。Aave 和 Venus 上的主动市场操作 (AMO) 允许比在 Lista DAO 平台上的直接调整更细致的利率控制。

* 竞争性借贷利率：目标是保持 Aave/Venus 上的 lisUSD 借贷利率低于 Lista DAO 平台上的利率，使其成为短期借款者的有吸引力的选择。
* 平台特定策略：
  * Aave/Venus：适合短期借款，具有更高的资本效率和动态利率。
  * Lista DAO 平台：最适合长期借款，提供通过社区治理确定的稳定利率。

## 收益和生态系统增长 <a href="#id-6a79" id="id-6a79"></a>

从 Aave 和 Venus 上的 lisUSD 存款产生的收益，包括利息和代币奖励，将回流到 Lista DAO 的财政部。这些资金将支持生态系统增长，资助提案，并加强社区奖励。

## 第一阶段实施细节：

* 初始铸币上限：250万 lisUSD
* 支持的平台：Venus 和类似的 DeFi 协议
* 为 Venus 核心池铸造 lisUSD
* 支持本金和利息的存款和提款
* 必要时能够销毁铸造的 lisUSD
* 利润和奖励检索机制，确保 DAO 直接从借贷平台活动中受益。
