# 路线图

## 2026 上半年 <a href="#h-key-highlights" id="h-key-highlights"></a>

## 引言 <a href="#h-introduction" id="h-introduction"></a>

Lista DAO 非常自豪地展示我们2026年上半年的路线图。基于我们在2025年的里程碑，我们将继续将Lista建设成一个全面的DeFi生态系统，该系统将传统金融与区块链创新桥接起来。Lista将发布几个最受需求的新产品，并完善我们现有的产品阵容，从流动性质押平台演变为涵盖借贷、交易、RWA和预测市场的多方面金融平台。

## 智能借贷和智能交换扩展 <a href="#h-smart-lending-and-smart-swap-expansion" id="h-smart-lending-and-smart-swap-expansion"></a>

Lista计划在2026年最重要的一步是扩展智能借贷/交换。这两个产品于2025年底推出，代表了Lista致力于提供更全面的DeFi服务的承诺和扩展为领先DeFi场所的雄心。

这个扩展计划包括三个关键组成部分：

* **超越现有市场**：包括更多资产，如（更多的）稳定币、包装资产和其他相关价值的代币。这些代币代表着具有最小价格波动性的巨大交易机会 - 正是DeFi用户需要的类型的基础设施。
* **整合DEX聚合器**：引入聚合器，从其他主要DEX中获取更大的交易流量，为我们的整个社区提供最佳的流动性和利率选项。
* **扩展到以太坊主网**：尽管gas成本较高，以太坊主网仍然是DeFi流动性的中心，拥有[超过700亿美元的总锁定价值](https://defillama.com/chain/ethereum)。这种多链方法伴随着另一个关键举措：在Smart Swap中引入更多资产，超出Stableswap模型，并引入DEX聚合器。

通过这些举措，Lista将为两个网络的用户提供更多选项。智能借贷/交换将成为交易者、流动性提供者和任何希望最大化其加密资产的人的首选选项。

## RWA：桥接TradFi和DeFi <a href="#h-rwa-bridging-tradfi-and-defi" id="h-rwa-bridging-tradfi-and-defi"></a>

自2023年以来，RWA领域经历了爆炸性增长，代币化的国库券、公司债务和其他传统资产的链上价值超过170亿美元。像Centrifuge、Ondo和Maple这样的协议表明，对产生收益的现实世界资产的需求很强，特别是随着DeFi收益的压缩和机构兴趣的增长。

通过在2025年12月启动我们的RWA市场，Lista进入了一个价值数万亿美元的全球市场。通过Lista，用户将能够购买美国国库券和抵押贷款义务，无需任何KYC要求。在2026年，我们的计划包括：

* **引入更多具有稳定收益的优质资产**：债券支持的抵押品、公司债券等。
* **为我们的RWA代币引入更多实用程序**：RWA代币不应该闲置在您的钱包中。它们应该并且将能够在我们的RWA区域用作您的贷款和更多的抵押品。
* **优化收益策略**：随着稳定的链下收益，Lista将制定出链上和链下产品的最佳组合，为我们的用户提供最优化的收益流。

这将把链下资产转化为生产性的链上构建块，解决RWA当前的一个限制：除了持有代币和从单一来源收集收益之外，没有足够的选项。

## 开创链上信用借贷 <a href="#h-pioneering-on-chain-credit-lending" id="h-pioneering-on-chain-credit-lending"></a>

Lista 2026年路线图中最具创新性的元素之一是探索链上信用借贷。

传统的DeFi借贷协议，如Aave和Compound，需要至少125%的抵押率，实际上限制了那些已经持有大量加密资产的人的信用访问。信用借贷可以通过进入一个潜在巨大的市场，使DeFi信用访问民主化。

这些贷款的工作方式非常简单：

* **无需抵押**：一切都是基于信用的。您做出的每一个良好的财务决定都会提高您的信用评分。
* **小额，短期**：几百美元用于紧急支付？有了良好的信用，您将在几秒钟内得到它。只需在您的工资到账时几天内还款。
* **高利息**：信用贷款往往有更高的违约率，因此，供应商的利率也更高。

市场机会巨大：仅美国就有[超过1.3万亿美元的循环信用额度](https://www.federalreserve.gov/releases/g19/current/)，加密可以通过更高的资本效率和全球可访问性来捕获。

话虽如此，Lista将一步一个脚印地谨慎行事，向基于我们专有信用评估系统选择的地址提供小额信用贷款。通过全面的链上行为分析，我们的目标是为那些有良好记录的人提供更多的流动性选项，同时尝试为流动性提供者提供最佳回报。

## 预测市场：通过预测吸引社区 <a href="#h-prediction-markets-engaging-community-through-forecasting" id="h-prediction-markets-engaging-community-through-forecasting"></a>

在短短一年多的时间里，链上预测市场的受欢迎程度激增，已发展成为一个价值数十亿美元的行业，仅在2024年美国选举周期期间，像[Polymarket这样的平台就处理了超过30亿美元的交易量](https://www.theblock.co/post/324357/us-presidential-winner-prediction-market-crosses-3-billion-on-polymarket-a-day-before-the-election)。随着领先的预测协议如Polymarket和Kalshi筹集了数十亿美元级别的投资，如此多的关注集中在这个新兴行业的巨大潜力上。

通过将预测市场直接整合到我们的借贷平台，Lista将创造包括以下内容的新颖参与机会：

* **对某些事件的结果进行投票**：用户将有机会对热门事件进行集体投票，而不是进行个人预测。
* **低风险，稳定收益的预测策略**：通过我们合作的预测市场，Lista将为我们的用户提供从全新来源赚取稳定收入的机会。
* **增强收益流**：对于非常高调的事件，用户甚至有机会用Lista的一部分收入预测其结果，让每个人都有机会在其资本上赚取更多收益。

预测市场的成功表明，设计良好的协议和池可以在事件结果上取得显著的准确性，往往超过专家预测和传统民意调查。通过引入这一垂直领域，Lista将利用集体智慧，同时创造额外的平台实用性和社区参与向量。

## 统一借贷体验 <a href="#h-unified-lending-experience" id="h-unified-lending-experience"></a>

Lista在2025年4月推出了其借贷产品，取得了惊人的成功 - 几个月内达到19.9亿美元的历史最高TVL，在BNB链上的借贷协议中排名第二。Lista的另一个旗舰产品，抵押债务头寸（CDP），允许用户存入各种资产（BTC、ETH、BNB等）并借用Lista的稳定币lisUSD。凭借近10亿美元的历史最高TVL，Lista CDP在过去几年中稳步增长。

虽然这两个产品各自以直观和良好管理的方式运作，但它们本质上在做同一件事：去中心化、无需许可的C2C借贷。随着它们与Lista一起成长，每个社区成员都必须面对一个关键的痛点：在多个界面之间的用户体验碎片化。

在2026年，Lista将从深层技术统一开始，而不仅仅是UI聚合，将这些产品在智能合约层面进行整合。抵押、借贷/借款、清算和管理将全部合并，以创造无缝的一站式借贷产品。当基础设施设置好后，Lista还将启动UI改造，将借贷、CDP和信用借贷整合到一个页面。

这种方法反映了许多加密平台的做法，它们已经从一个简单产品的界面扩展到综合金融中心。通过这种整合，Lista将提供统一的借贷体验，抽象出复杂性的同时最大化用户选项。

Lista始终认为直观的设计和无缝的用户流程与技术创新一样重要。通过这种持续的完善努力，Lista将整合以前分散的功能到更加统一的工作流程中，同时也将打磨我们的UI设计和网站结构。

## 建立在坚实的基础上 <a href="#id-35e6" id="id-35e6"></a>

在2024年，Lista DAO达到了几个变革性的里程碑，巩固了其在BNB链上的主要地位。该协议在总锁定价值（TVL）上经历了指数级增长，从1.1086亿美元飙升至11亿美元，同比增长896.92%。这一成就使Lista DAO成为BNB链上第三大协议。

此外，LISTA代币在币安上的推出引入了veLISTA治理模型，赋予社区积极参与协议决策的权力。安全仍然是首要任务，通过实施多重预言机架构、TimeLock治理和持续审计来保护用户资产并确保运营完整性。

## 2025年赋能LISTA代币持有者 <a href="#id-0427" id="id-0427"></a>

Lista DAO将从2025年开始专注于通过veLISTA治理框架赋能LISTA代币持有者。这个系统将授予持有者重要的特权，包括lisUSD借款利息折扣、延迟清算和其他独家福利。

分层奖励系统将激励更深入的veLISTA承诺，为代币持有者提供更高的排放增益。此外，引入公开市场代币赎回将有助于稳定LISTA的价值，并通过创建一个可持续和平衡的代币经济来促进长期增长。
