# 路线图

## 在坚实的基础上建设 <a href="#id-35e6" id="id-35e6"></a>

到2024年，Lista DAO在BNB链上达到了几个变革性的里程碑，巩固了其作为主要参与者的地位。该协议在锁定总价值（TVL）上经历了指数级增长，从1.1086亿美元飙升至11亿美元，标志着年初至今增长896.92%。这一成就使Lista DAO成为BNB链上第三大协议。

此外，LISTA代币在币安的推出引入了veLISTA治理模型，赋予社区积极参与协议决策的权力。安全仍然是首要任务，通过实施多Oracle架构、TimeLock治理和持续审计来保护用户资产并确保运营完整性。

## 2025年赋能LISTA代币持有者 <a href="#id-0427" id="id-0427"></a>

Lista DAO将在2025年初通过veLISTA治理框架专注于赋能LISTA代币持有者。这一系统将授予持有者重要的特权，包括lisUSD借款利息折扣、延迟清算和其他独家福利。

分层奖励系统将激励更深入的veLISTA承诺，为代币持有者提供更高的排放增强。此外，开放市场代币赎回的引入将有助于稳定LISTA的价值并促进长期增长，通过创建一个可持续和平衡的代币经济。

## 推进三大核心支柱 <a href="#cde6" id="cde6"></a>

Lista DAO将继续优先发展其三个关键支柱，以此为基础：

### Lista 1.0: lisUSD扩展 <a href="#d836" id="d836"></a>

协议将推动lisUSD在多个链上的采用，针对缺乏可靠稳定币选项的生态系统。为了进一步吸引用户和流动性，Lista DAO计划推出新的lisUSD产品，如固定借款利率和有限的免费借款期，以提升用户参与度和协议收入。

### Lista 2.0: 流动性质押增强 <a href="#id-4efd" id="id-4efd"></a>

对slisBNB的增强将专注于优化节点运营，通过创建更小、可扩展的节点进一步去中心化，并加强治理。这些改进将提高性能和可访问性，确保为用户提供稳健和可持续的质押体验。

### Lista 3.0: 扩展BNBFi生态系统 <a href="#eda7" id="eda7"></a>

随着Lista DAO扩展其跨链集成，BNBFi生态系统（slisBNBx）将在2025年经历显著增长。通过与StakeStone和Solv等协议合作，Lista DAO将从其他区块链（包括以太坊）引入质押资产到BNB。这一举措将为用户解锁参与Launchpool、Megadrop和Hodler Airdrop等计划的机会，提供超过30% APR的竞争性收益。

为进一步增强生态系统，Lista DAO将推出跨链金库产品，旨在捕获多个网络的激励和空投。此外，将推出推荐系统以吸引影响者和KOLs，推动用户增长并加速采用。

作为其领导DeFi空间策略的一部分，Lista DAO将专注于将更多DeFi原语整合到其生态系统中。这些“乐高积木”将包括扩展的抵押品选项、先进的流动性质押解决方案和创新的流动性机制。通过构建一个模块化和互操作的生态系统，Lista DAO旨在为用户提供更大的灵活性，同时解锁资本效率和优化的新机会。

## 2025年下半年路线图

### **1. USD1中心领导**

将Lista建设成为USD1最大的链上中心，目前Lista Lending中已有超过7000万美元，目标是将TVL增加到超过1亿美元，并通过扩展集成和收益优化提供竞争性的供应APY。

### **2. Lista CDP为PancakeSwap**

计划在第三季度启动，使PancakeSwap LP头寸作为lisUSD铸造的抵押品，提高资本效率。

### **3. RWA产生收益的稳定币**&#x20;

在第三季度初推出以USD1为基础的、由美国国债和企业债券支持的稳定币。

### **4. lisUSD增长**&#x20;

通过我们的产生收益的稳定币和LP铸造产品，将lisUSD铸造量从6320万增加到8000万（增长26.6%）。

### **5. 类似Fluid的DEX**

在第四季度推出一个借贷和AMM集成的DEX，将Lista Lending抵押品转换为LP头寸。

### **6. 固定利率借贷扩展**

扩展Lista CDP以提供固定利率和固定期限的借贷产品，时间表待确认。
